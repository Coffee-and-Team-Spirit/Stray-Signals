Build Notes

Known Bugs

  • Cannot save during the drink mini game — loading will be corrupted and cause UI errors.
  • A long player name (> 50 characters) overflows the dialogue box.
  • Occasionally very fast/double clicking during the game will cause the UI to repeat or become unplayable until reload.

Certification

I confirm that I have tested this build on Windows & MacOS and it runs as submitted. The core mechanic is functional and a player can start, play, and reach the end of our narrative mystery by crafting drinks for each encounter over the course of 5 days. Known issues are documented in the Build Notes above.
